# Login Form – README

## How to Run

1. Open the project in NetBeans IDE.
2. Locate the file `LoginForm.java`.
3. Right-click `LoginForm.java` → Run File
   (or press Shift + F6).
4. The Login window will open in the center of the screen.

---

## Login Credentials

Use the following details to log in

```
Username admin
Password 123
```

## Notes

 Both fields are required.
 Enter credentials exactly as shown (case-sensitive).
 If incorrect details are entered, login will fail.
