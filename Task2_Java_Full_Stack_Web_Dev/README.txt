LOGIN & SIGNUP SYSTEM (PHP + MySQL)

HOW TO RUN PROJECT:

1. Copy project folder to htdocs (xampp/lampp)
2. Start Apache & MySQL
3. Open phpMyAdmin
4. Create database: user_auth
5. Import database.sql
6. Open browser:
   http://localhost/TASK2-AUTH/signup.php

FEATURES:
- User Registration
- Login Authentication
- Password Hashing
- Session Management
- Protected Dashboard
- Secure Logout
